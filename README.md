# All Door API/Web Server

## 설명
새로운 공간 공유 플랫폼 **All-Door Back-EndAPI / WEB Server**입니다.

## 사용 라이브러리

- Express

- Mongoose

- Redis

- Jquery

## 사용방법
`npm install`

`npm start`

## Config

#### 문자 알람 설정 (청기와랩)

`ALL_DOOR_SMS_SENDER=[ SMS SENDER]`

`ALL_DOOR_SMS_APP_ID=[ SMS APP ID ]`

`ALL_DOOR_SMS_API_KEY=[ SMS API KEY ]`

#### 이미지 파일 저장 위치

`ALL_DOOR_IMAGE_PATH= [이미지 파일 저장 위치 ]`
